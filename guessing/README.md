# Random number guessing game

Goal: Create a game that asks the user to guess a random number.

## Steps

1. Create a random number
2. Ask the user to guess the number we have.
    1. Note: A random number can be between 0 to 2^31
    2. We have a way to ask the user for a number, see coins.c
4. If the number is same, the user have won and the game finishes
5. If the number is not the same, then tell the user if their guess is bigger or smaller than the random number
6. Keep the game going until the user guesses the number correctly
7. When the game is complete, tell the user how many guesses they used

## Requirements

1. The game must follow the steps
2. The program must compile and run before showing
3. The program must be checked into git
